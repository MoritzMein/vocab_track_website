Aufbau der /schulen Seite

Welche Struktur überzeugt die Schulleitung?

A

Problem → Lösung → Kontakt

Klassischer Verkaufs-Aufbau: erst das Problem benennen, dann VocabTrack als Antwort präsentieren.

Seitenstruktur
① Hero: „Vokabellernen neu gedacht — für Ihre Schule"
② Problem: Warum das alte System nicht funktioniert
③ Lösung: Was VocabTrack anders macht (3 Punkte)
④ Vertrauen: DSGVO-Badge + Kostenlos in Pilotphase
⑤ Kontaktformular
B

Vorteile → Vertrauen → Kontakt

Direkt zu den Vorteilen — kein langes Einleiten. Schulleitung weiß was das Problem ist, sie wollen wissen was VocabTrack bietet.

Seitenstruktur
① Hero: „VocabTrack für Schulen — kostenlos in der Pilotphase"
② 4 Vorteile: Zeitsparend / KI-gestützt / DSGVO / Fortschritt einsehbar
③ So funktioniert's: 3 Schritte (Lehrer → Schüler → Einblick)
④ Pilotphase-Banner: Jetzt kostenlos starten
⑤ Kontaktformular
C

Kurz & direkt — alles auf einen Blick

Schulleiter haben wenig Zeit. Kompakte Seite: Hero mit Kernaussage, 3 Stichpunkte, Kontaktformular. Fertig.

Seitenstruktur
① Hero: Headline + 2 Sätze was VocabTrack ist
② 3 Bullet-Points: DSGVO ✓ / Kostenlos ✓ / KI-gestützt ✓
③ Kontaktformular
